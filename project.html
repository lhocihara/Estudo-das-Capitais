O Projeto
O que é?

O “Estudo das Capitais” é um site que tem como objetivo ajudar as crianças a aprender sobre a geografia brasileira, de uma forma divertida e interativa, a partir de um quiz para descobrir quais são as capitais de cada estado, com um sistema de tempo e pontuação.


Por quê?

Visando facilitar o ensino da disciplina de Geografia, este projeto foi criado na base "aprender brincando", onde estudos comprovam que este método é eficaz, a recompensa por pontos bem como a sensação de conquista gera uma sensação não massante que uma leitura pode acarretar.


Com tudo isso em mente, temos uma boa forma de ensinar, educar e incentivar o desenvolvimento comum dessa disciplina.
